# EaseStay-Streamlined-Booking

EaseStay is a full-stack booking application designed to provide users with a seamless and intuitive experience for booking accommodations. Built using the MERN stack (MongoDB, Express.js, React.js, Node.js), EaseStay offers features similar to popular platforms like Airbnb, enabling users to search, book, and manage their stays effortlessly.

## <a name="quick-start">ğŸ¤¸ Quick Start</a>

Follow these steps to set up the project locally on your machine.

1. Clone the repository:

```bash
git clone https://github.com/your-username/stay-ease.git
npm install

```
2. Install dependencies:
```bash
npm install
```
3. Set up environment variables:
Create a .env file in the root directory and define the following variables:
```bash
MONGO_URL=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
S3_ACCESS_KEY=your_aws_s3_access_key
S3_SECRET_ACCESS_KEY=your_aws_s3_secret_access_key
```
4.Start the server:
```bash
npm start
```
5.Start the client:
```bash
cd client
npm start
